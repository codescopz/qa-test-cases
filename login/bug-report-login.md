## Bug Report - Login

### Bug 01 — Mensagem de erro genérica

**Descrição:**  
Ao informar senha incorreta, o sistema exibe mensagem pouco clara.

**Passos para reprodução:**
1. Acessar a tela de login
2. Informar e-mail válido
3. Informar senha inválida
4. Clicar em "Entrar"

**Resultado esperado:**  
Exibir mensagem "Senha incorreta"

**Resultado atual:**  
Exibe mensagem "Erro ao autenticar"

**Prioridade:** Média  
**Severidade:** Média
