# Projeto de Testes Funcionais

Repositório criado para estudo e prática em Quality Assurance (QA),
com foco em testes funcionais manuais.

O projeto contém:
- Casos de teste
- Cenários de teste (BDD)
- Relatórios de bugs

As funcionalidades são organizadas por pastas,
seguindo boas práticas de documentação e versionamento com GitHub.
